# Plugin Installation Standards

When installing any new plugin or library, you **MUST** check the `context7` MCP server first to retrieve up-to-date documentation and implementation examples.

## Procedures
1. Resolve Library ID: Use `mcp_context7_resolve-library-id` with the package name.
2. Query Documentation: Use `mcp_context7_query-docs` to understand usage and best practices.
