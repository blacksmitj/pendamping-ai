# UI Development Standards

When creating or modifying UI components, you **MUST** use the `shadcn` MCP server to ensure consistency and speed.

## Procedures
1. Search Components: Use `mcp_shadcn_search_items_in_registries` for finding components.
2. Get Examples: Use `mcp_shadcn_get_item_examples_from_registries` to see implementation demos.
3. Add Components: Use `mcp_shadcn_get_add_command_for_items` to get the installation command.
