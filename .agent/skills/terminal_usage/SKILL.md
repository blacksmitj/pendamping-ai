# Terminal Usage Standards

All terminal commands **MUST** be executed within the **Windows PowerShell** environment.

## Rules
1. Shell: Always assume a PowerShell context.
2. Paths: Use Windows-style paths (backslashes) or ensure cross-platform compatibility.
3. Multi-line: Use PowerShell line continuation (` `) if necessary.
